## 📖 项目简介

**Review Helper** 是一款专为学术研究者设计的智能文献综述生成工具。它通过整合Google Scholar搜索、AI关键词提取、智能文献筛选和自动综述生成等功能，将原本需要数周甚至数月的文献综述工作，压缩到几个小时即可完成。

### 🎯 我们为什么做这个？

**痛点：**
- 📚 **文献搜索耗时**：手动在Google Scholar上搜索和筛选相关文献，需要花费大量时间
- 🔍 **关键词提取困难**：如何从研究主题中提取有效的搜索关键词，往往需要反复尝试
- 📝 **文献筛选繁琐**：需要逐篇阅读文献摘要，判断是否与研究主题相关
- ✍️ **综述撰写耗时**：整理文献信息、归纳观点、撰写综述内容，需要大量人工工作
- 💾 **数据管理混乱**：文献信息、中间结果分散在各个地方，难以统一管理
- 🎭 **虚假文献问题**：大模型直接生成的综述往往存在虚假文献，引用不存在的文献或捏造文献信息
- 💰 **工具收费昂贵**：市面上现有的文献综述工具大多需要付费订阅，增加了研究成本

**解决方案：**
- 🤖 **AI驱动的自动化流程**：从关键词提取到综述生成，全程AI辅助
- 🔗 **无缝集成Google Scholar**：直接搜索和提取学术文献信息，确保所有文献真实存在
- ✅ **基于真实文献的综述生成**：只使用从Google Scholar实际搜索到的文献，杜绝虚假文献问题
- 📊 **智能项目管理**：所有数据集中管理，支持多项目并行工作
- 🎨 **简约大气的界面**：科研风格设计，专注于内容本身
- 🆓 **完全免费开源**：基于MIT许可证，无需付费订阅，降低研究成本

## ✨ 核心功能

### 模块一：文献搜索与关键词提取

- **AI关键词提取**：输入研究主题，AI自动提取相关搜索关键词
- **Google Scholar集成**：自动搜索并提取文献信息（标题、作者、年份、摘要、引用数等）
- **智能去重**：自动合并搜索结果并去除重复文献
- **灵活配置**：可设置关键词数量、每关键词文献数、年份范围、引用数要求等

### 模块二：文献整理与筛选

- **AI智能筛选**：基于文献内容，AI自动判断文献是否与研究主题相关
- **多种导入方式**：
  - 从模块一自动导入搜索结果
  - 通过PDF文件导入（AI自动提取文献信息）
  - 通过论文名称搜索导入（Google Scholar精准搜索）
- **完整编辑功能**：可编辑文献的所有信息（标题、作者、年份、来源、摘要等）
- **灵活管理**：支持删除文献、批量选择、Excel导出等

### 模块三：文献综述自动生成

- **基于真实文献生成**：只使用从Google Scholar实际搜索到的真实文献，确保所有引用文献都真实存在，杜绝虚假文献问题
- **AI自动撰写**：基于选定的真实文献，AI自动生成结构清晰、逻辑严密的文献综述
- **准确引用**：自动在引用位置标记文献序号 `[文献序号]`，所有文献信息可追溯
- **可编辑内容**：支持对生成的综述进行编辑和优化
- **Word导出**：导出为Word兼容的HTML格式，便于进一步编辑

### 项目管理

- **多项目支持**：可同时管理多个研究项目
- **自动保存**：所有操作自动保存，防止数据丢失
- **状态追踪**：清晰显示项目进度（模块1/2/3）
- **数据持久化**：所有项目数据本地存储，支持离线使用

## 🚀 快速开始

### 环境要求

- **Node.js** >= 14.0.0
- **npm** >= 6.0.0
- **DeepSeek API Key**（用于AI功能，可在[DeepSeek官网](https://www.deepseek.com/)申请）

### 安装步骤

1. **克隆或下载项目**
   ```bash
   git clone <repository-url>
   cd 文献项目
   ```

2. **安装依赖**
   ```bash
   npm install
   ```

3. **启动应用**
   ```bash
   npm start
   ```

### 首次使用

1. **创建新项目**
   - 在主页面点击"新建项目"
   - 输入项目名称
   - （可选）输入项目简介作为提醒

2. **配置API Key**
   - 在模块页面输入您的DeepSeek API Key
   - API Key会自动保存到项目中

3. **开始使用**
   - 按照模块顺序完成文献搜索、整理和综述生成

## 📖 使用指南

### 模块一：文献搜索

1. **输入综述主题**
   - 在"综述主题输入"框中输入您的研究主题
   - 例如："深度学习在图像识别中的应用"

2. **配置搜索参数**
   - **关键词数量**：AI将提取多少个关键词（默认6个）
   - **每关键词文献数**：每个关键词搜索多少篇文献（默认10篇，最大19篇）
   - **文献年份**：可设置最小年份，筛选较新的文献
   - **引用数量**：可设置最小引用数，筛选高影响力文献

3. **提取关键词**
   - 点击"分析关键词"按钮
   - AI会自动分析并提取相关关键词
   - 可以手动编辑、添加或删除关键词

4. **搜索文献**
   - 首次搜索需要登录Google Scholar并完成验证
   - 验证完成后，系统会自动搜索所有关键词
   - 搜索结果会自动合并并去重

5. **查看和确认**
   - 查看最终文献列表
   - 可以导出为Excel进行备份
   - 点击"确认完成"保存并进入模块二

### 模块二：文献整理

1. **查看文献列表**
   - 自动加载模块一的所有文献
   - 显示文献的完整信息

2. **补充文献（可选）**
   - **通过PDF导入**：
     - 点击"选择PDF文件"
     - 选择PDF文件后点击"导入PDF"
     - AI会自动提取文献信息
   - **通过论文名称搜索**：
     - 输入论文标题
     - 点击"搜索并导入"
     - 系统会在Google Scholar中搜索并导入

3. **AI筛选文献**
   - 点击"开始AI筛选"
   - AI会分析每篇文献，判断是否与研究相关
   - 筛选结果会显示推荐理由
   - AI推荐的文献会自动选中

4. **手动选择**
   - 可以手动勾选或取消勾选文献
   - 支持全选、全不选、仅选AI推荐等操作

5. **编辑文献信息**
   - 点击文献右侧的"✏️"按钮编辑文献信息
   - 可以修改标题、作者、年份、来源、摘要等所有字段

6. **删除文献**
   - 点击文献右侧的"🗑️"按钮删除不需要的文献

7. **确认完成**
   - 选择完成后，点击"确认完成"进入模块三

### 模块三：综述生成

1. **查看选定文献**
   - 系统会显示所有在模块二中选择的文献
   - 显示文献的标题、作者、年份、来源等信息

2. **生成综述**
   - 点击"开始生成综述"
   - AI会分析所有选定的文献
   - 自动生成结构清晰、逻辑严密的文献综述
   - 生成完成后会显示在文本框中

3. **编辑综述**
   - 可以直接在文本框中编辑综述内容
   - 所有修改会自动保存

4. **导出Word文档**
   - 点击"导出Word文档"
   - 系统会生成Word兼容的HTML文件

5. **确认完成**
   - 检查无误后，点击"确认完成"
   - 项目状态会更新为"已完成"

## 💡 使用技巧

### 提高搜索效率

1. **关键词优化**：如果AI提取的关键词不够准确，可以手动添加更精确的关键词
2. **筛选条件**：合理设置年份和引用数要求，可以快速筛选出高质量文献
3. **批量导入**：如果已有PDF文件，可以批量导入，系统会逐个处理

### 优化筛选结果

1. **AI筛选后手动检查**：AI筛选结果仅供参考，建议手动检查每篇文献
2. **编辑文献信息**：如果文献信息不完整，可以手动补充
3. **删除无关文献**：及时删除不相关的文献，保持列表整洁

### 改善综述质量

1. **选择高质量文献**：优先选择引用数高、年份较新的文献
2. **编辑AI生成的综述**：AI生成的综述是基础，建议根据实际需求进行编辑
3. **多次生成对比**：如果对结果不满意，可以删除后重新生成

## ⚠️ 注意事项

1. **API Key安全**：请妥善保管您的DeepSeek API Key，不要泄露给他人
2. **Google Scholar验证**：首次搜索需要登录Google Scholar并完成验证，后续搜索会自动进行
3. **数据备份**：建议定期导出Excel文件进行备份，项目数据存储在`projects`文件夹中
4. **网络连接**：需要网络连接才能使用AI功能和Google Scholar搜索
5. **PDF解析**：PDF导入功能需要安装`pdf-parse`包，如果未安装，系统会使用简化版本
